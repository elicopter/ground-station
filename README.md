# Elicopter Ground Station [![Slack Status](https://elicopter-slackin.herokuapp.com/badge.svg)](https://elicopter-slackin.herokuapp.com/)

Ground station for Elicopter core flight controller.

## Getting Started

Install dependencies:
```
mix deps.get
npm install
```

Run locally:
```
mix phoenix.server
```
